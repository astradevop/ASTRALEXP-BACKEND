---
description: Build Android and iOS apps using EAS CLI
---

This workflow sets up and initiates the build process for the Android platform using Expo Application Services (EAS).

// turbo
1. Run the build command for Android to initialize credentials and start building:
   `npx eas-cli@latest build --platform android`

2. Follow any interactive prompts (if any) to provide necessary permissions.
3. Monitor the build progress via the links provided in the output.
